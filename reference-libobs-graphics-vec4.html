#include <graphics/vec4.h>
struct vec4

Two component vector structure.

float vec4.x

X component

float vec4.y

Y component

float vec4.z

Z component

float vec4.w

W component

float vec4.ptr[4]

Unioned array of all components

  • dst – Destination
